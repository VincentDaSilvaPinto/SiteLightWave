Le logiciel


LightWave est un logiciel qui permet de générer en temps réel un effet visuel dépendant du rythme d’un morceau de musique, tout en vous offrant la possibilité de personnaliser ce logiciel parce le biais de fonctionnalitées. Pour le réaliser nous avons utilisé le langage C# dont le code est disponible. De plus une maquette de la version idéale est disponible.
Le logiciel, les maquettes,le code et les notices sont disponibles dans l'onglet "téléchargement"
